# Serviko Admin Panel

## 📌 Overview

The **Serviko Admin Panel** is a minimal and efficient dashboard designed to manage core platform operations such as customers, providers, categories, bookings and approvals. It focuses on simplicity, clarity and fast administrative workflows.

---

## ✨ Key Features

### 📊 Dashboard

* Overview of key metrics
* Quick insights into platform activity
* Summary cards for fast monitoring

---

### 👨‍🔧 Provider Management

* View all service providers
* Approve or reject provider requests
* Access provider details
* Manage provider status

---

### 🗂️ Category Management

* Create and manage service categories
* Assign icons and colors
* Organize services efficiently

---

### 📋 Request Handling

* Review incoming requests
* Accept or reject submissions
* View detailed request information

---

### ✅ Approval System

* Dedicated approval workflows
* Popup-based confirmation actions
* Status tracking (Approved / Pending / Rejected)

---
